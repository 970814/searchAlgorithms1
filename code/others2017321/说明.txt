 文件名 起始行数 结束行数(不包括) 间距 y轴比例
 5个值，应该用空格隔开，如

2.键盘事件
case 'K': 是否绘制当前鼠标到当前编辑对象构成的线的1个以上的角度
case 'k': 是否绘制所有图形的每条线条的角度
case 'l': 是否绘制其拟合曲线函数
case 'b': 忽略 和 'l' 类似
case 32://space:(空格键) 切换编辑对线，切换到另外一个图进行编辑
case 'n': 新增加一个图进行编辑

case 'r': 画出整个图的特征线条
case 'R': 取消绘制整个图的特征线条
case 'f': 分段绘制整个图的特征线条，当进行图形匹配之前，必须先进行该操作,并且每次图形发生改变都要执行该操作
case 'F': 取消绘制整个图的分段特征线条,但是特征线条还是存在这个图中,也就是说按下f后,在按F,同样适用于图形匹配
case 'c': 对当前编辑的对象对其他所有的图进行搜索，匹配是否相似，最终结果在控制台最后一行输出，true or false
case 'C': 按下c键以后，相似部分会被红色矩形框框起来,如不想看到，可以按这个键
case 'A': 正反向添加一个图，输入图的名字即可,扩展名不必，且默认为.txt，改图必须存在data/ 中。
case 'a': 正向添加一个图，输入图的名字即可,扩展名不必，且默认为.txt，改图必须存在data/ 中。
3.鼠标右键弹出菜单事件:
1.前面几个复选框，有图像存在试试，试试选上，看看效果便知
2.在单选框 add 模式下，鼠标左键即可添加（对当前 编辑对象）一个新的点在鼠标当前指向位置
3.在单选框 remove 模式下，拖动鼠标绘制矩形，在范围内的点将会被删除，
4. normal 模式下，什么也不能干，但是有个好处，当不需要对图进行 编辑的时候，不会因为错误点击而影响了图原来的特征
5. 用select模式对点进行选择，也是拖动鼠标绘制矩形，在范围内的点将会被选中，
6.使用moveTo模式，必须先使用select模式，然后鼠标左击屏幕不放，对选中的进行整体移动
